# NumPy Data Explorer

## 📌 Project Overview

NumPy Data Explorer is a beginner-friendly Python project that demonstrates the core features of the NumPy library for efficient numerical computing and data manipulation. The project covers array creation, indexing, slicing, mathematical operations, reshaping, broadcasting, file handling, and performance comparison with standard Python lists.

## 🚀 Features

* Create NumPy arrays using different methods.
* Perform array indexing and slicing.
* Apply mathematical, statistical, and axis-wise operations.
* Reshape arrays and use broadcasting for efficient computation.
* Save and load NumPy arrays using `.npy`, `.npz`, and `.txt` formats.
* Compare NumPy performance with standard Python lists.

## 🛠️ Technologies Used

* Python 3.x
* NumPy

## ▶️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/NumPy-Data-Explorer.git
```

### Navigate to the project folder

```bash
cd NumPy-Data-Explorer
```

### Install NumPy

```bash
pip install numpy
```

### Run any Python file

```bash
python array_creation.py
```
